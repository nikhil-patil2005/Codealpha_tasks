# 🎮 Hangman Game

A simple console-based Hangman game built using Python. The player guesses a hidden word one letter at a time with a maximum of six incorrect attempts.

## Features
-  Random word selection
-  Letter-by-letter guessing
-  6 attempts to guess the word
-  Win and Game Over messages

## Technologies
- Python
- Random Module
- Loops
- Conditional Statements

## Run

```bash
python hangman.py
```
